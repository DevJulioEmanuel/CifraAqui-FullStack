# 🎸 CifraAqui

> Plataforma fullstack inspirada no Cifra Club para cadastro, busca e visualização de cifras musicais.

---

## 📌 Sobre o projeto

O **CifraAqui** é uma aplicação fullstack que permite aos usuários cadastrar músicas, criar e consultar cifras, favoritar conteúdos e explorar informações enriquecidas sobre as músicas — tudo com autenticação segura e uma interface moderna.

O projeto nasceu como um desafio técnico pessoal com o objetivo de praticar e demonstrar boas práticas de desenvolvimento tanto no back-end quanto no front-end.

---

## 🧱 Tecnologias

### Back-end
- **Java 21** + **Spring Boot 3**
- **Spring Security** com autenticação via **JWT**
- **Spring Data JPA** + **PostgreSQL**
- **Bean Validation** e **Lombok**
- **SpringDoc / Swagger** para documentação da API
- Integração com a **API do Deezer**

### Front-end
- **React** + **TypeScript**
- **Vite**
- **Axios** para comunicação com a API
- **React Router** para navegação

### Infraestrutura
- **Docker** + **Docker Compose** para containerização de toda a aplicação

---

## 🔧 Funcionalidades previstas

### 🔐 Autenticação
- Cadastro e login de usuários
- Autenticação via token JWT

### 🎵 Músicas
- Listagem e busca de músicas por nome ou artista
- Detalhes da música com capa e preview via Deezer
- Ranking de músicas mais acessadas

### 🎸 Cifras
- Visualização de cifras
- Criação de cifras (usuário autenticado)
- Transposição de acordes

### ❤️ Extras
- Favoritar músicas e cifras
- Player de preview de 30s integrado ao Deezer
- Documentação interativa via Swagger
